# getXiguaVideo
一键下载西瓜视频脚本

## 依赖
python 3.7.2

pip 19.0.2

## 安装

```
pip3 install -r requirements.txt  
```

## 食用

1.从西瓜视频PC端打开对应视频的网址，把视频地址替换到下面的参数上
```
videoUrl = "https://www.ixigua.com/i6702721188671521293/"  

```
2.执行下述命令下载即可。

```
python3 getXiguaVideo.py
```
